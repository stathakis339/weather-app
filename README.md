# weather-app
A simple weather application built with vanilla JavaScript, HTML and CSS.
Users can search for a city and get real-time weather information from a public API.

Live >  https://stathakis339.github.io/weather-app/

This project was created to practice:
- API requests with Fetch
- Async / Await
- DOM manipulation
- Error handling
- Clean UI structure

This repository is part of my front-end learning portfolio.
